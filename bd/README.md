# Backend

Backend-приложение для хакатона, разработано на ASP.NET Core с использованием C# 8.

## Основные особенности

- **Аутентификация и авторизация**: Реализована собственная система аутентификации на основе JWT-токенов и cookie. Поддерживаются токены доступа (access token) и обновления (refresh token).
- **Подтверждение Email**: При регистрации пользователь получает письмо с подтверждением адреса электронной почты.
- **Сброс пароля**: Возможность восстановления пароля через Email.
- **Отправка Email**: Работа с электронной почтой реализована через background job, использующий Hangfire для асинхронной отправки писем.
- **Конфигурация**: Почти все настройки вынесены в конфигурационный файл. Подробнее о настройках можно прочитать[здесь](configuration.md).
- **Валидация конфигурации**: При запуске приложения все параметры конфигурации проходят валидацию с помощью `IValidateOptions`. При обнаружении неверных значений приложение завершает работу с ошибкой.
- **Управление секретами**: Секреты (пароли, ключи API и т.д.) хранятся в отдельных файлах в папке `secrets` и передаются в контейнеры Docker через механизм secrets.

## Технологии

- **ASP.NET Core (C# 8)**
- **PostgreSQL** — основная база данных
- **Redis** — для кэширования и хранения временных данных (например, токенов подтверждения)
- **Hangfire** — для выполнения фоновых задач (отправка Email)
- **Docker** — для контейнеризации и деплоя
